# Post formats that work (2026)

All English. Link ALWAYS in first reply, never the main post.

## 1. Build-in-public update (single post)

```
Shipped today: {one-line what}.

- {concrete detail with a number}
- {concrete detail with a number}
- {cost or time — honesty sells}

{one-line lesson or surprise}
```

## 2. Numbered thread (bookmark bait)

- Tweet 1 = hook: surprising number/claim + "Here's how/why: 🧵"
- Tweets 2-6 = one numbered point each, 2-4 lines, concrete
- Last tweet = takeaway + soft CTA ("Bookmark this" / question)
- [REPLY] = link to the article

## 3. Screenshot + story (single post)

Native image of the actual thing (dashboard, terminal, article page) + 3-5
lines of story. Screenshots of real numbers outperform stock everything.

## 4. Hot take (single post)

One earned opinion, no hedging, ends with a question. Only post takes you can
defend with your own experience.

## Anti-patterns (never)

- Link in main post
- "🚀🔥 GAME CHANGER" language
- Threads longer than 8 tweets
- Posting without replying to comments in the first hour
- Engagement-bait polls with no substance
